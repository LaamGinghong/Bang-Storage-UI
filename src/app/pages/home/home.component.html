Bang Storage UI of Angular

这里是广州棒谷科技股份有限公司仓储物流组基于NG-ZORRO的组件二次封装。

+

特性

  • 基于NG-ZORRO的二次封装，与NG-ZORRO兼容。
  • 开箱即用的高质量Angular组件。
  • 使用TypeScript构建，提供完整的类型定义文件。

支持环境

现代浏览器和IE11以上（需要polyfills）。

当前版本

支持Angular版本

目前支持Angular^7.0.0版本

安装

我们强烈推荐使用官方的@angular/cli工具链辅助进行开发，在实际项目开发中，它可以很好的满足TypeScript代码的构建、调试、代理、打包部署等一系列工程化的需求。 
    
    $ ng new PROJECT_NAME
    $ cd PROJECT_NAME
    $ ng add ng-zorro-antd
与此同时，我们也强烈推荐使用Yarn来管理你的依赖。如果你不知道如何安装Yarn,请点击 这里。 
    
    $ ng set packageManager=yarn
