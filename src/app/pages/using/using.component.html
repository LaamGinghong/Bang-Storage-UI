快速上手

Bang Storage UI 致力于为棒谷前端开发工程师提供愉悦的开发体验。

官方指南假设你已了解关于 HTML、CSS 和 JavaScript 的中级知识，并且已经完全掌握了 Angular 及配套设施的正确开发方式。 如果你刚开始学习前端或者 Angular ，将框架作为你的第一步可能不是最好的主意 —— 掌握好基础知识再来吧！

我们为你准备了一套教程，能够帮助你快速上手Bang Storage UI。

第一个在线实例

实际项目开发中，你会需要对 TypeScript 代码的构建、调试、代理、打包部署等一系列工程化的需求。 我们强烈建议使用官方的@angular/cli工具链辅助进行开发。与此同时，我们强烈建议你使用Yarn作为你的包管理器， 因此我们的例子都将使用Yarn。下面我们用一个简单的例子来说明。

1.安装脚手架工具

使用@angular/cli前，务必确认Node.js已经升级到v11.0或以上，强烈建议升级至最新版本的@angular/cli。 如果你想了解更多CLI工具链的功能和命令，建议访问Angular CLI了解更多。 

    
    $ yarn global add @angular/cli

2.创建一个项目

在创建项目之前，请确保@angular/cli已被成功安装。

执行以下命令，@angular/cli会在当前目录下创建一个名为PROJECT_NAME的文件夹，并自动安装好依赖。

      
      $ ng new PROJECT_NAME

3.初始化配置

进入项目文件夹，执行以下命令引入Bang Storage UI。

    
    $ yarn add bang-storage-ui

4.引入NG-ZORRO和echarts

由于Bang Storage UI是基于NG-ZORRO的二次封装，因此你需要执行以下命令。

    
    在根模块中
    import {{'{NgZorroAntdModule, NZ_I18N, zh_CN}'}} from 'ng-zorro-antd';
    import {{'{BangStorageUIModule}'}} from 'bang-storage-ui';

    @NgModule({{'{'}}
     imports:[BangStorageUIModule,NgZorroAntdModule],
     providers: [{{'{provide: NZ_I18N, useValue: zh_CN}'}}]
    {{'}'}})

    export class AppModule{{'{'}}
    {{'}'}}

    在angular.json中
     {{'{'}}
       "projects":{{'{'}}
        "PROJECT_NAME":{{'{'}}
         "architect":{{'{'}}
          "build":{{'{'}}
           "options":{{'{'}}
            "assets":[
             "src/favicon.ico",
             "src/assets",
               {{'{'}}
                 "glob": "**/*",
                 "input": "./node_modules/@ant-design/icons-angular/src/inline-svg/",
                 "output": "/assets/"
               {{'}'}}
             ],
             "scripts": [
               "./node_modules/echarts/dist/echarts.js"
             ]
            {{'}'}}
          {{'}'}}
        {{'}'}}
      {{'}'}}
    {{'}'}}

5.开发调试

一键启动调试，运行成功后进入页面。

    
    $ yarn start -o

6.构建和部署

    
    $ ng build --prod

入口文件会构建到dist目录中，你可以自由部署到不同环境中进行引用。
