Button 按钮

用于开始一个即时的操作。

何时使用

标记了一个（或封装一组）操作命令，响应用户点击行为，触发相应的业务逻辑。

代码演示

按钮有四种类型：默认按钮、主按钮、危险按钮、警告按钮、成功按钮、信息按钮。

按钮类型










添加disabled属性即可让按钮处于不可用状态，同时按钮样式也会改变。

不可用状态










添加storageGhost属性后，幽灵按钮将其他按钮的内容反色，背景变为透明，常用在有色背景上。

幽灵按钮


按钮有大、中、小三种尺寸。

通过设置storageSize为largesmall分别把 按钮设为大、小尺寸。若不设置storageSize,则尺寸为中。

按钮尺寸










添加storageLoading属性即可让按钮处于加载状态。

加载中状态

API

[storage-button]Directive

通过设置Button的属性来产生不同的按钮样式，推荐顺序为： storageType-> storageGhost-> storageSize-> storageLoading-> disabled
